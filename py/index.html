Skulpt 是一个在浏览器中运行 Python 代码的 JavaScript 库，支持大部分 Python 3 语法与部分标准库功能，适用于编写与执行轻量级、纯Python代码，特别适合于编程教学、交互式学习、在线代码编辑与简单应用。受限于浏览器环境，它不支持C扩展库、本地文件访问、直接网络通信或多线程/进程，不适合复杂的、依赖底层硬件或特定库的编程任务。
